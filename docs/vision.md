# msh プロダクトビジョン

## 一言で言うと

**Rust で書いた、速くて軽く、誰でも触れる対話型シェル。**

Zsh ユーザーが移行しやすく、Fish を超える UX を、Bash 100% 互換を目指さずに実現する。

---

## 4 つの柱

### 1. 速度（Speed）

| 観点 | 目標 |
|---|---|
| 起動時間 | Fish 並み、Bash より速い |
| 入力応答 | キー入力から画面反映まで < 16ms（60fps 相当） |
| 補完生成 | 体感即時（< 50ms） |
| パイプライン | 外部コマンド起動のオーバーヘッドを最小化 |

### 2. 省メモリ（Memory）

| 観点 | 目標 |
|---|---|
| アイドル時 RSS | < 5 MB（対話セッション中） |
| 起動時 | < 8 MB（`-c` モード） |
| 履歴・補完キャッシュ | 上限付き、設定で調整可能 |

### 3. 互換性（Compatibility — Zsh 同等を目指す）

**100% Bash 互換は目指さない。Zsh と同等の「日常使い・移行しやすさ」を網羅する。**

| レベル | 内容 | 目標 |
|---|---|---|
| L1 対話 | エイリアス、cd、export、パイプ、リダイレクト、ジョブ制御 | ✅ Phase 2 完了 |
| L2 構文 | `&&` `\|\|` `;` `$()` `${var}` コマンドチェイン・置換 | Phase 5 |
| L3 スクリプト | `if` / `for` / `while` / `case`、関数、配列（基本） | Phase 5–6 |
| L4 移行 | 一般的な `.bashrc` / `.zshrc` の 80%+ がそのまま動く | Phase 5 完了条件 |
| L5 高度 | 連想配列、process substitution `<()`、全 POSIX/Bash 拡張 | 非目標（必要時は `bash` 呼び出し） |

**Zsh 同等の意味（具体例）**

- 既存のエイリアス・関数・export がそのまま使える
- パイプライン・リダイレクト・バックグラウンドジョブが期待通り動く
- 一般的な CI / デプロイスクリプトの大部分が `-c` や `source` で実行できる
- `.bashrc` を丸ごと `source` しても致命的に壊れない（未対応構文は明示的にエラー）

**あえて目指さないもの**

- Bash 5.x の全拡張、GNU 固有オプションの完全再現
- 100% の POSIX テストスイート合格
- Fish 独自構文のエミュレーション

### 4. 使いやすさ（UX — Fish 超え、初心者ファースト）

**Bash / Zsh / Fish のどれよりも、初見でも迷わない体験をデフォルトで提供する。**

| カテゴリ | msh の方針 | 既存シェルとの差 |
|---|---|---|
| 発見性 | 空 Enter で使える操作ヒント、Tab で候補 + 説明 | Fish より能動的 |
| エラー | 何がダメか + 直し方 + 類似コ mand 提案（日本語/英語） | Bash/Zsh より親切 |
| 補完 | ファジー + インラインサジェスト + 文脈（git 等） | Fish 同等以上 |
| ハイライト | 入力中リアルタイム、パイプ/リダイレクトも色分け | 全競合よりリッチ |
| 設定 | 1 ファイル TOML、設定不要の sensible defaults | Zsh+Oh-My-Zsh より簡単 |
| 学習曲線 | 5 分で Tab・履歴・補完が使える | 全競合より緩やか |

**初心者向け UX の具体像**

- コマンドが見つからない → 「`ls` ですか？」と提案
- パイプの使い方が分からない → エラーに例文を添える
- 設定ファイル不要で「開いた瞬間から気持ちいい」
- 専門用語（ジョブ、ファイルディスクリプタ等）を避け、平易な言葉で説明

---

## ターゲットユーザー

1. **Zsh / Bash から移行したい開発者** — 互換性を保ちつつ UX を上げたい層
2. **ターミナル初心者** — IDE 的な操作感で怖さを減らしたい層
3. **Fish ユーザー** — UX は好きだがスクリプト互換・速度に不満がある層
4. **リソース制約環境** — コンテナ・CI・リモート VM で軽いシェルが欲しい層

---

## 非目標（やらないこと）

- **Bash 100% 互換** — Zsh 同等が現実的な上限
- データ指向シェル（Nushell の領域）
- GUI ターミナルエミュレータの開発
- Oh-My-Zsh 規模のプラグインエコシステムを Phase 1 で追求

---

## 成功指標（KPI）

| 指標 | v0.5 (Phase 5) | v1.0 (Phase 6) |
|---|---|---|
| 起動時間 | < 30ms | < 10ms |
| アイドル RSS | < 8 MB | < 5 MB |
| 補完レイテンシ p95 | < 100ms | < 30ms |
| `.bashrc` 互換率 | **80%+** | 90%+ |
| 基本操作の学習時間 | **5 分以内** | 3 分以内 |
| UX 主観評価 | Fish 同等以上 | **全競合より高い** |

    Fish  │
          │

    Bash  │

        低 UX ────────────────── 高 互換性
